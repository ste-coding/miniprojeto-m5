Sobre

Com o aumento do consumo de eletrônicos, há uma quantidade cada vez maior de dispositivos descartados, muitos dos quais contêm materiais tóxicos que podem causar danos significativos ao meio ambiente e à saúde humana se não forem gerenciados corretamente.
Assim, este projeto tem como objetivo criar uma API para integrar um sistema voltado para a solução prática do crescente problema do descarte inadequado de resíduos eletrônicos. Desenvolvido como parte do Módulo 4 da formação Fullstack do Instituto Programadores do Amanhã.

API

A e-Waste API busca facilitar o processo de descarte de resíduos eletrônicos, permitindo que empresas e indivíduos localizem facilmente pontos de coleta onde esses itens podem ser deixados para reciclagem ou descarte seguro. A API serve como um intermediário, conectando doadores de lixo eletrônico com centros de coleta certificados, garantindo que os resíduos sejam tratados de forma adequada.
